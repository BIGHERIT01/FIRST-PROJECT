📊 Finance Transactions Data Analysis

🔍 Overview

This project analyzes a financial transactions dataset using Python to uncover trends, patterns, and key insights. It involves data cleaning, exploration, and visualization to support better understanding of financial performance.

---

🎯 Objectives

- Clean and prepare raw financial data
- Analyze transaction trends and patterns
- Visualize financial metrics using charts
- Draw meaningful insights from the dataset

---

🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

📁 Dataset

- Source: Finance Transactions Dataset (Excel file)
- Size: Approximately 1000 rows and 14 columns
- Key Features:
  - Transaction_Type
  - Amount
  - Net_Amount
  - Tax_Amount
  - Transaction_Date
  - Approval_Status
  - Sector
  - Department
  - Analyst
  - Currency

---

📊 Key Insights

- Transaction amounts vary across different transaction types
- Some sectors generate higher net amounts and taxes than others
- Monthly transaction volume shows noticeable trends over time
- Most transactions are approved, with fewer rejected or pending
- Top analysts handle the highest transaction volumes

---

📈 Visualizations

- Bar chart: Total Amount by Transaction Type
- Grouped bar chart: Net Amount vs Tax Amount by Sector
- Line chart: Monthly Transaction Volume
- Pie chart: Approval Status Distribution
- Horizontal bar chart: Top 5 Analysts by Total Amount
